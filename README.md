# 🚀 Minimalist CV

Un portafolio web de alto rendimiento construido con **Astro**, diseñado para ser ultraligero y fácil de mantener mediante la importación de datos estructurados en JSON.

---

## 🛠️ Stack Tecnológico

- **Framework:** Astro 
- **Lenguajes:** HTML5, CSS3
- **Data Engine:** JSON dinámico para la gestión de contenido
- **Entorno:** Desarrollado sobre WSL (Ubuntu)

---

## 🎯 Características Principales

### 📂 Data-Driven
Todo el contenido (experiencia, proyectos, skills) se gestiona desde un único archivo `cv.json`.

### ⚡ High Performance
Optimización de carga por debajo de 1 segundo, cumpliendo con métricas de **Core Web Vitals al 100%**.

### 📱 Responsive Design
Adaptable a cualquier dispositivo con un diseño minimalista y profesional.

### 🔄 Workflow Moderno
Control de versiones con Git y despliegue automatizado.
